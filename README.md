# Four-balls

Implemented in two methods: 
* Procedurally 
* OOP

# Working
Four Balls move from left to right

Ball 1 is at 1/5 height from the top, with the speed 1 units per frame  
Ball 2 is at 2/5 height from the top, with the speed 2 units per frame  
Ball 3 is at 3/5 height from the top, with the speed 3 units per frame  
Ball 4 is at 4/5 height from the top, with the speed 4 units per frame
